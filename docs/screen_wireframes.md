# NutriLens 텍스트 와이어프레임

> PRD 사용자 흐름(Step 1~6)에 대응하는 7개 화면의 구조 정의
> 범례: `[ ]` 버튼, `< >` 입력필드, `{ }` 칩/태그, `---` 구분선

---

## 화면 1: 스플래시 / 온보딩
**대응 흐름:** PRD Step 1 (앱 최초 진입)

```
┌─────────────────────────────────┐
│                                 │
│                                 │
│        🥬 NutriLens             │
│                                 │
│   건강을 담은 스마트 냉장고     │
│   내 몸에 맞는 식재료만 골라,  │
│   가장 먼저 먹어야 할 것을 알려드려요 │
│                                 │
│                                 │
│      [ 시작하기 →  ]           │
│                                 │
│   이미 계정이 있으신가요?       │
│        [ 로그인 ]               │
│                                 │
└─────────────────────────────────┘
```

**핵심 요소**
- 로고 + 슬로건: 서비스 가치 제안 1줄 요약
- CTA 버튼: "시작하기" → 회원가입 화면으로 이동
- 보조 링크: 기존 사용자를 위한 로그인 경로

## 화면 2: 회원가입
**대응 흐름:** PRD Step 1 (계정 생성)

```
┌─────────────────────────────────┐
│  ← 뒤로                        │
│                                 │
│   계정 만들기                   │
│   건강 정보는 안전하게 보호됩니다 │
│                                 │
│   이메일                        │
│   < example@email.com        >  │
│                                 │
│   비밀번호                      │
│   < ••••••••                 >  │
│   ✓ 8자 이상  ✓ 숫자 포함      │
│                                 │
│   비밀번호 확인                 │
│   < ••••••••                 >  │
│                                 │
│      [ 다음 단계 →  ]          │
│                                 │
│   ───── 또는 ─────             │
│   [ 🍎  Apple로 계속하기  ]    │
│   [ G   Google로 계속하기 ]    │
│                                 │
└─────────────────────────────────┘
```

**핵심 요소**
- 인라인 비밀번호 정책 안내 (입력 중 실시간 표시)
- 소셜 로그인 옵션 (Apple, Google)
- "다음 단계" → 질환 선택 화면으로 이동

## 화면 3: 질환 선택 (온보딩 Step 1/2)
**대응 흐름:** PRD Step 2 (건강 프로필 설정 - 질환)

```
┌─────────────────────────────────┐
│  ← 뒤로          [건너뛰기]    │
│                                 │
│  ●━━━━━○  Step 1 / 2           │
│                                 │
│   앓고 계신 질환을              │
│   선택해 주세요                 │
│   (해당하는 것을 모두 선택)     │
│                                 │
│   {당뇨}   {고혈압}            │
│   {고지혈증}  {신장질환}       │
│   {통풍}                        │
│                                 │
│   ───────────────────────────   │
│   {없음 / 해당 없음}           │
│                                 │
│   ℹ️ 선택한 질환에 따라        │
│      해로운 식재료를            │
│      자동으로 걸러드립니다.     │
│                                 │
│      [ 다음 →  ]               │
│                                 │
└─────────────────────────────────┘
```

**핵심 요소**
- 진행바 (Step 1/2): 온보딩 진행 상황 시각화
- 칩 UI: 복수 선택 가능, 선택 시 색상 강조
- "없음" 칩: 선택 시 다른 칩 선택 해제 (단독 선택)
- 건너뛰기: 이 단계를 패스하고 알레르기 선택으로 이동

## 화면 5: 메인 / 냉장고 스캔
**대응 흐름:** PRD Step 3 (냉장고 스캔)

```
┌─────────────────────────────────┐
│   NutriLens          👤 [내 정보] │
│                                 │
│  ┌───────────────────────────┐  │
│  │                           │  │
│  │    📸 냉장고를 촬영하면   │  │
│  │    식재료를 자동으로       │  │
│  │    인식합니다             │  │
│  │                           │  │
│  └───────────────────────────┘  │
│                                 │
│      [  📷 카메라 스캔  ]      │
│      [  🖼️  갤러리 선택  ]     │
│                                 │
│   ─── 또는 직접 입력 ───       │
│   < 식재료 이름 검색...     >  │
│                                 │
│   최근 등록 식재료              │
│   🟠 우유 (D-1)  🟡 당근 (D-4) │
│   ⚪ 양파 (기한없음)            │
│                                 │
│ ─────────────────────────────── │
│  🏠 홈   📷 스캔   📋 레시피   │
└─────────────────────────────────┘
```

**핵심 요소**
- 카메라 스캔: 냉장고 사진 촬영 → OCR/AI 식재료 인식
- 갤러리 선택: 기존 사진에서 인식
- 직접 입력: 검색으로 수동 추가
- 최근 등록 목록: 골든타임 상태 배지 미리 표시 (🔴기한초과, 🟠골든타임, 🟡주의)
- 하단 네비게이션: 홈 / 스캔 / 레시피

## 화면 6: 스캔 결과
**대응 흐름:** PRD Step 4 (필터링 결과 확인)

```
┌─────────────────────────────────┐
│  ← 뒤로       스캔 결과        │
│                                 │
│  인식된 식재료: 8개             │
│  ✅ 안전: 5개  ⛔ 주의: 3개   │
│                                 │
│  ─── 사용 가능 ────────────── │
│  🔴 우유         [기한초과]    │
│     유통기한: 3/10 (3일 초과)  │
│                                 │
│  🟠 달걀         [골든타임]    │
│     유통기한: 3/14 (D-1)       │
│                                 │
│  🟡 당근         [주의]        │
│     유통기한: 3/17 (D-4)       │
│                                 │
│  ⚪ 양파         [여유]        │
│     유통기한: 4/10 (D-28)      │
│                                 │
│  ─── 사용 불가 ─────────────  │
│  🚫 소금         ⛔질환 제한: 고혈압│
│  🚫 간장         ⛔질환 제한: 고혈압│
│  🚫 새우         ⛔알레르기: 새우  │
│                                 │
│      [ 레시피 추천 받기 →  ]   │
│                                 │
└─────────────────────────────────┘
```

**핵심 요소**
- 요약 카운트: 안전/주의 식재료 수 한눈에 파악
- 골든타임 배지 (🔴🟠🟡): 색상으로 긴급도 시각화
- 차단 배지 (🚫): 차단 사유 함께 표시 (알레르기/질환)
- "레시피 추천 받기" CTA → 레시피 추천 화면으로 이동

## 화면 7: 레시피 추천
**대응 흐름:** PRD Step 5~6 (레시피 추천 + 구매 연결)

```
┌─────────────────────────────────┐
│  ← 뒤로       레시피 추천      │
│                                 │
│  🟠 지금 써야 할 재료           │
│  달걀(D-1)  우유(기한초과)     │
│                                 │
│  ─── 추천 레시피 ────────────  │
│  ┌─────────────────────────┐   │
│  │ 🍳 달걀 스크램블         │   │
│  │ ⏱ 10분  👤 1인분        │   │
│  │ 재료: 달걀✓ 버터✗ 소금✗ │   │
│  │      (✗ 부족한 재료)    │   │
│  │  [ 레시피 보기 ]        │   │
│  │  [ 🛒 부족 재료 구매 ]  │   │
│  └─────────────────────────┘   │
│                                 │
│  ┌─────────────────────────┐   │
│  │ 🥛 달걀찜               │   │
│  │ ⏱ 20분  👤 2인분        │   │
│  │ 재료: 달걀✓ 우유✓ 소금✗ │   │
│  │  [ 레시피 보기 ]        │   │
│  │  [ 🛒 부족 재료 구매 ]  │   │
│  └─────────────────────────┘   │
│                                 │
│  ─── 오늘 먹지 않으면? ──────  │
│  우유 버리면 약 1,800원 손해 💸 │
│                                 │
│      [ 다른 레시피 더 보기 ]   │
│                                 │
└─────────────────────────────────┘
```

**핵심 요소**
- 골든타임 재료 하이라이트: 상단에 별도 섹션으로 강조
- 레시피 카드: 소요 시간, 인분, 보유 재료(✓)/부족 재료(✗) 표시
- 부족 재료 구매 링크: 쿠팡/마켓컬리 등 외부 쇼핑 연결
- 낭비 비용 안내: 버렸을 때의 손해 금액 표시로 행동 동기 부여

## 화면 흐름 요약

```
[스플래시] → [회원가입] → [질환 선택] → [알레르기 선택]
                                                  ↓
                    [메인/스캔] ←────────────────┘
                        ↓
                   [스캔 결과]
                        ↓
                  [레시피 추천]
```

| 화면 | PRD Step | 핵심 API |
|------|----------|----------|
| 1. 스플래시 | - | - |
| 2. 회원가입 | Step 1 | Supabase Auth |
| 3. 질환 선택 | Step 2 | `GET /api/diseases` |
| 4. 알레르기 선택 | Step 2 | `GET /api/allergens` |
| 5. 메인/스캔 | Step 3 | 카메라/갤러리 |
| 6. 스캔 결과 | Step 4 | `POST /api/recommend` |
| 7. 레시피 추천 | Step 5~6 | `POST /api/recommend` |
